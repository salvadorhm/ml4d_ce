# Webapp para minería de datos

## Webapp versión

versión 0.1.0

## Descripción

Herramienta de apoyo para realizar minería de datos basada en:

* numpy
* pandas
* statsmodels
* matplotlib
* sklearn
* seaborn
* scipy
* web.py
